**Kasap Pro** is an all-in-one, offline-first Mobile Point of Sale (mPOS) and Financial Management ecosystem tailored for modern butcher shops, delicatessens, and retail meat businesses.

## 🚀 The Solution
Traditional butcher shops often struggle with fragmented systems for barcode scanning, weight scale integration, and debt tracking. **Kasap Pro** unifies these operations into a single, high-performance Android application built on **.NET MAUI**, operating entirely locally via **SQLite** for maximum reliability.

### ✨ Key Features
* **Modern SaaS UI/UX:** Minimalist dashboard design with a clean aesthetic, supporting both Light and Dark modes.
* **Advanced Scale Integration:** Dynamically parses 13-digit barcodes from industrial weight scales (Bizerba, Mettler, etc.) to extract weight and calculate prices instantly.
* **Wholesale & Debt Ledger:** Built-in "Veresiye" (debt) tracking system and wholesale module for managing B2B relationships.
* **Financial Analytics:** Real-time data visualization using animated charts to track revenue, expenses, and net profit.
* **Automated Reporting:** Generates multi-tab Excel reports using `ClosedXML` for easy sharing via WhatsApp or Email.
* **Setup Wizard:** An intuitive onboarding process to customize store branding, scale prefixes, and administrative roles.
