**CepKod POS** is a comprehensive, offline-first Mobile Point of Sale (POS) and Financial Management application. 

This repository contains the customized deployment for **Öz Biga Et** (a real-world meat & deli business), demonstrating a robust software solution for complex daily retail operations, wholesale tracking, expense management, and advanced financial reporting.

## 🚀 The Challenge & The Solution
Local businesses often struggle with fragmented systems—using separate tools for barcode scanning, scale (weight) calculations, wholesale tracking, and debt logging. **CepKod POS** unifies all these processes into a single, high-performance Android application built entirely on **.NET MAUI**. It operates 100% locally via **SQLite**, ensuring zero downtime even without an internet connection.

---

## ✨ Core Modules & Features

### 🛒 1. Advanced POS & Barcode Engine
* **Real-Time Scanning:** High-speed barcode detection via device camera (`ZXing.Net.MAUI`).
* **Hardware Feedback:** Haptic vibration and audio feedback (`Plugin.Maui.Audio`) for a true POS machine experience.
* **Dynamic Scale Integration:** Automatically parses proprietary 13-digit barcodes generated by industrial weight scales (Bizerba, Mettler) to extract weight (grams) and calculate prices dynamically.

### 🏢 2. B2B Wholesale & Debt Management (Veresiye)
* **Wholesale Module:** Dedicated pipeline for B2B sales (Restaurants, Hotels). Automatically tags the purchasing company to the invoice.
* **Digital Ledger:** Built-in debt (Veresiye) tracking system. Logs customer balances, partial payments, and generates detailed account statements.

### 📈 3. Financial Analytics & Excel Reporting
* **Visual Dashboards:** Integrated **Microcharts & SkiaSharp** to render beautiful, animated Donut Charts showing the daily top-selling products by revenue.
* **Net Cash Calculation:** An automated accounting engine that calculates Gross Profit, subtracts Daily Expenses (Rent, Bills), accounts for debts, and outputs the exact **Net Cash** in the register.
* **Excel Export:** Uses `ClosedXML` to generate multi-tab `.xlsx` reports (Retail, Wholesale, Expenses, Profit Summary) and shares them instantly via WhatsApp or Email.

### 🔐 4. Enterprise Security (RBAC)
* **Role-Based Access Control:** * `Patron (Owner)`: Full access to expenses, user management, and net profit.
    * `Müdür (Manager)`: Access to reports, inventory, and sales.
    * `Kasiyer (Cashier)`: Locked strictly to the POS sales screen.
* **Cryptographic Security:** Passwords are mathematically hashed before entering the database.
